Projects

MiTM ATTACK (USING ARP AND DNS SPOOFING)

In this project, I developed and executed a demonstration of a Man-in-the-Middle (MITM) attack using ARP (Address Resolution Protocol) and DNS (Domain Name System) spoofing techniques. The primary objective was to showcase the vulnerabilities in network communication and raise awareness about the importance of securing data transmission.

  • Techniques Used:
    • Utilized ARP spoofing to manipulate device communication.
    • Employed DNS spoofing to redirect legitimate domain requests.

DDoS ATTACK

In this project, I simulated a Distributed Denial of Service (DDoS) attack on a virtual machine. The main goal was to pretend to attack a computer system in different ways, see how it affected the system, and explore how we can protect against these kinds of attacks.

  • Techniques Used:
      Implemented the TCP SYN method, a tactic often exploited in DDoS attacks. This method involves overwhelming a system by systematically bombarding it with connection requests that are intentionally left incomplete.
